# Evently App

[![Flutter Version](https://img.shields.io/badge/Flutter-3.22-blue)](https://flutter.dev)
[![Firebase](https://img.shields.io/badge/Firebase-Backend-orange)](https://firebase.google.com)
[![Provider](https://img.shields.io/badge/State%20Management-Provider-green)](https://pub.dev/packages/provider)

An event planning and management app that allows users to create, update, favorite, and delete events across multiple categories. Built with Firebase backend and Provider state management.

## Features

| Feature | Description |
|---------|-------------|
| 👋 **Onboarding** | 3-screen introduction for first-time users |
| 🔐 **Authentication** | Email/Password + Google Sign-In |
| 📝 **Create Events** | Add events with title, description, date, time, location, and category |
| 📂 **Categories** | 10 categories (Sport, Birthday, Meeting, Gaming, Workshop, Book Club, Exhibition, Holiday, Eating, All) |
| ❤️ **Favorites** | Mark/unmark events as favorites |
| ✏️ **Update Events** | Edit existing event details |
| 🗑️ **Delete Events** | Remove events permanently |
| 👤 **User Profile** | Update profile, toggle theme (Light/Dark), change language (English/Arabic) |
| 🌍 **Localization** | Full English/Arabic support |
| 🎨 **Theming** | Light and dark mode |

## EventListProvider (Key Methods)

| Method | Purpose |
|--------|---------|
| `getAllEvents(uId)` | Fetch all events, sort by date |
| `getFilterEventsList(uId, context)` | Filter by category |
| `changeSelectedIndex(index, uId, context)` | Switch between All / category |
| `updateEventListFavStatus(event, uId, context)` | Toggle favorite status |
| `getFavoriteList(uId)` | Get all favorite events |
| `updateEventDetails(event, uId, context)` | Edit event |
| `deleteEvent(event, uId, context)` | Delete event |

## Setup Instructions

### Prerequisites
- Flutter SDK installed
- Firebase project created
- Android app registered in Firebase Console

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/IslamRamzy444/project_2_evently.git
   cd project_2_evently
2. **Get dependencies**
   ```bash
   flutter pub get
3. **Configure Firebase**
   - Download `google-services.json` from Firebase Console
   - Place it in `android/app/`
   - Add debug SHA-1 fingerprint to Firebase Console (for Google Sign-In)
3. **Run the app**
   ```bash
   flutter run
